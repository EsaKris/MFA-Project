About the Project

Chapter Three: Implementation

This project is a comprehensive implementation of a Multi-Factor Authentication (MFA) system. It includes features like OTP via SMS, Email, and Google Authenticator, along with a secure and responsive user interface.

The system is built using Django for the backend and Tailwind CSS for the frontend. It ensures secure user authentication and provides real-time notifications for security events.

The project also includes an admin dashboard for managing users and security logs, as well as exportable reports in CSV and PDF formats.
